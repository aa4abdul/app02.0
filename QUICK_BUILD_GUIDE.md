# 🚀 APK Banane Ka Complete Guide

## Method 1: AppsGeyser (5 Minutes) ⭐ RECOMMENDED
1. https://appsgeyser.com pe jayein
2. "Create App" → "Website" template
3. Apna HTML code paste karein
4. App name aur icon set karein
5. APK download karein - FREE!

## Method 2: GitHub Actions (Automatic)
1. GitHub account banayein
2. Yeh HTMLRunner folder upload karein
3. Actions tab mein APK automatically build hogi
4. Download from artifacts

## Method 3: Android Studio (Professional)
1. Android Studio download: https://developer.android.com/studio
2. "Open" → HTMLRunner folder select karein
3. Build → Build APK(s)
4. APK location: app/build/outputs/apk/debug/
